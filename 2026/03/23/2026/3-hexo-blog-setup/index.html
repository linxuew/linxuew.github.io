简介

本博客基于 Hexo 搭建，使用 Butterfly 主题，部署在 GitHub Pages 上。Hexo 是一款快速、简洁且高效的静态博客框架，使用 Markdown 写作，一键生成静态页面，非常适合技术博客。

本文记录从零搭建到日常使用的完整流程。

常用命令

命令 说明
hexo new post <title> 新建文章
hexo new page <name> 新建页面（如 tags、about）
hexo clean 清除缓存和 public 目录
hexo generate 生成静态文件
hexo server 启动本地预览服务
hexo deploy 部署到远端
hexo g -d 生成并部署（常用组合）
